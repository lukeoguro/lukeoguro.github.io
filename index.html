Resume Learn About Edamame

Edamame is an open-source load testing framework for real-time collaboration apps that use both HTTP and WebSockets.

It features:

  • A stream processing pipeline that ingests 2MM messages / sec
  • A k6 plugin written in Go that emits WebSockets metrics

With Edamame, users can run distributed load tests from a self-hosted Kubernetes cluster on AWS with one command.

It also comes with a React UI that allows users to easily manage load tests.
